# 收集一个学校实体的写法

你的任务:对一个目标学校实体,通过 `/gather-entity-aliases` skill 把候选清单里指向它的所有写法登记到 `entity_aliases`。

## 项目级约束:学校 entity = 学历教育归属

talent-graph 的学校 entity 服务于员工的学历教育归属——下游回答"该员工是否在 X 学校接受过学历教育"。**挂母校名但本身不属于该校学历教育的主体一律跳过**——不写 alias。

具体边界:

**学历教育归属判定**(补充 SKILL 的通用规则):

- 下属研究院 / 学院 / 书院 / 研究生院:挂本校名 + 官网列明归属 → 归本校(例:深圳研究生院 / 光华管理学院)
- 改名 / 曾用名 / 院系调整前身:官网列明为本校前身 → 归本校

**不归属**(跳过):

- 附中 / 附小 / 附属实验学校等基础教育——独立的中小学阶段,不属于本校学历教育
- 附属医院——医疗机构,不承担学历教育
- 科技园 / 创业园 / 产业园——运营主体是园区公司,非校区
- 控股旗下上市公司 / 关联基金会(如"清华同方"、"清华紫光")——控股关系,非办学
- 培训机构、短期研修班——非学历教育
