# 定义一个学校域名单标签

你的任务:对一个学校域名单标签,通过 `/define-tag` skill 完成 tag 字段注册 + 标准学校实体清单挂载。

## 项目级约束:动态榜单必须锁年份

QS 前 100 / 泰晤士前 100 这类每年都变的榜单类标签,任务参数 `[消歧说明]` 必须明确锁哪一年(如"2025 QS 前 100");没锁年份直接视为清单不成立 → 终止任务等待消歧。code 也要带年份后缀(如 `qs_top100_2025`),不要直接写 `qs_top100`——明年榜单一变,不带年份的 code 会让同一 tag 的含义随时间漂移。

## 完成时输出一行结果

```
tag: <code> (<name>)
状态: created | already_exists
本轮动作: 挂入 N 家 / 解绑 M 家 / 无变更
依据: <一句话来源,如"教育部 2024 双一流名单官网">
```

请开始处理:

```
/define-tag school <标签名>
```
